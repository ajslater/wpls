# wpls (Wallpaper List)

List active wallpaper image paths on all macOS desktops.

## Install

```sh
pip instal wpls
```

## Use

```sh
$ wpls
Desktop 1 /Users/aj/Pictures/Wallpaper/Chromecast/4088949046_5d094cb2e2_o.jpg
Desktop 2 /Users/aj/Pictures/Wallpaper/National Geographic/2008/0915wallpaper-11_1280.jpg
```
